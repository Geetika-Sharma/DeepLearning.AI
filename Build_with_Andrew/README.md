## Telling AI What you want
### Funny Birthday Card Message Generator
#### Prompt Text
```
Create a web page to help me write birthday cards where I can give a person’s name, age and hobby and it gives me back a funny message.

Add a festive title and colours.

Display the card on the right side, and make it look like the inside of a birthday card.

Add a fun title at the top of the page. And instead of replacing the old cards, keep them visible and stack them underneath the new ones.
```

### Ping Pong Game
<img width="500" height="500" alt="image" src="https://github.com/user-attachments/assets/cb1938fa-9cfe-4710-b272-0ce96cc58ca9" />

#### Prompt Text
```
Build me a table tennis game as a single HTML file. User plays against computer and moves the paddle with the arrow keys.

Add 3 difficulty levels to choose from and allow the user to specify the number of points require	d to win. At the bottom, keep the scores.

Make the playground green, the paddles beige, and the ball white. Insert this image into the background:
https://raw.githubusercontent.com/https-deeplearning-ai/lc-build-with-andrew-platform/main/img/tabletennis43.jpg
```
## Final Project
### Prompt Text
```
Create a web page as a single HTML file to help me write motivational quote for the day where I can give a person’s name, age and how they are feeling and it gives me back a motivational message

Add a serene title and colours

Display the card on the right side, and make it look like the inside of a angel card.

Add an inspirational title at the top of the page. And instead of replacing the old cards, keep them visible and stack them underneath the new ones.
```
